SE Basic IV Buffy

An open source firmware for the Chloe 280SE, Timex 2000 series and ZX Spectrum compatible microcomputers.

Online user guide

Over 25 new features in SE Basic IV Buffy

  • BORDER sets the color pair in text mode.
  • CALL sets the required value of HL' on return.
  • CONTROL-shift enables block graphics, UDGs, and more to be entered directly.
  • Cursor up and down in an INPUT line.
  • Embedded codes are not permitted in functions.
  • Faster drawing in graphics mode.
  • Faster editor.
  • Fully commented source code.
  • Graphics commands can now use the entire display: PLOT 0, -16: PRINT POINT (0,-16).
  • Improved support for legacy microdrive syntax.
  • Inline color codes are ignored in text mode to prevent screen flicker.
  • Labels changed to Logan/O'Hara.
  • List a single line or a range of lines: LIST: LIST 10: LIST 10,: LIST ,100.
  • META-shift enables characters 32 to 255 to be entered directly.
  • Modular assembly listing.
  • NEW 48 performs a warm restart and sets compatibility mode.
  • NEW 128 performs a cold restart (clearing 128 RAM). For devices that force USR 0 mode.
  • Refactored code for improved readability.
  • ROM switching service routine.
  • SAVE now saves immediately without displaying a message and corrupting the screen.
  • SCREEN$ (x,y) support in 80-column mode.
  • SCROLL command for console style text handling in 32- or 80-column mode.
  • SPEED command sets CPU speed.
  • TRACE command for debugging.
  • Universal cursor.
